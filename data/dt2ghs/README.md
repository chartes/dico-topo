dt2ghs
===

Export [GHS](https://github.com/geoTirroirs/geoSnippets/tree/master/ghs) des données des DicoTopo.
Testé uniquement sur le département de l’Aisne (`DT02/DT02.xml`) avec [BaseX](http://basex.org/).
